## Use of AI assitance (ChatGPT)

During the development of this project, ChatGPT was used a support tool for:
-Understanding technical concepts
-Debugging and troubleshooting issues
-Improving code readability and structure

All implementation and design decisions were made and understood by the author.

### Areas where AI assistance was used

-Backend:
-routers/bookings.py
Assistance with understanding loop logic for handling multiple seat bookings.

-routers/movies.py 
Guidance on implementing poster uploads and file handling, as well as deletion logic stored images.

-routers/main.py
Help with implementing a catch-all route to correctly server the React frontend.

-Frontend:
-src/AdminPage.jsx
Assistance with debugging async fucntions (handlesubmit, handleShowtimeSubmit) and improving component logic.

-src/MovieSeatsPage.jsx 
Guidance on handling state and understanding the interaction between components.

-src/SeatGrid.css & SeatGrid.jsx 
Assistance with implementing seat selection behavior and visual feedback (e.g. marking selected seats).

-src/MoviesPage.jsx
Guidance on structuring and designing the UI to match my intended layout.
